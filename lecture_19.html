Lecture 19: Linear Regression & Extensions

1. Introduction

Linear regression is one of the most fundamental supervised learning algorithms. It models the relationship between a dependent variable Y and one or more independent variables X. The basic form is:

Y = β0 + β1X + ε

where β0 is the intercept, β1 is the slope, and ε is the error term.

Example: Predicting a student’s exam score (Y) based on study hours (X). If slope β1=5, then each extra hour of study increases the score by 5 points.

2. Six Models & Variations

Model 1: Simple Linear Regression

One independent variable (X). Example: Predicting house price from area (sq. ft).

Model 2: Multiple Linear Regression

Several features. Example: House price from area, number of rooms, and location.

Model 3: Polynomial Regression

Introduces higher powers of X. Example: Predicting car price depreciation curve.

Model 4: Ridge Regression

Applies L2 regularization to prevent overfitting. Useful when features are correlated.

Model 5: Lasso Regression

Applies L1 regularization to perform feature selection by shrinking some coefficients to zero.

Model 6: Elastic Net Regression

Combines L1 and L2 penalties. Example: Predicting health outcomes using many medical features.

3. Applications

  • Finance: Predicting stock returns.
  • Healthcare: Estimating disease risk from patient data.
  • Marketing: Sales forecasting based on ad spend.
  • Sports: Predicting player performance metrics.
