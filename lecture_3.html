Lecture 3: Probability & Statistics for Machine Learning

1. Probability Basics

Probabilities are used in Naive Bayes, HMMs, and Bayesian models.

Python Example: Probability Distribution


import numpy as np

# Probability of coin flips
flips = np.random.choice(['H', 'T'], size=1000)
p_heads = np.mean(flips == 'H')
print("P(Heads):", p_heads)

2. Statistics in ML

Mean, variance, and standard deviation help in feature scaling, normalization, and hypothesis testing.

Python Example: Descriptive Statistics


import numpy as np

data = np.array([2, 4, 6, 8, 10])
print("Mean:", np.mean(data))
print("Variance:", np.var(data))
print("Std Dev:", np.std(data))

3. Application: Gaussian Distribution

Many ML models (like Gaussian Naive Bayes) assume features follow a normal distribution.


import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import norm

x = np.linspace(-5, 5, 100)
y = norm.pdf(x, 0, 1)  # mean=0, std=1

plt.plot(x, y)
plt.title("Gaussian Distribution (Normal Curve)")
plt.show()
