Lecture 17: Introduction to Machine Learning

Machine Learning (ML) is a field of Artificial Intelligence (AI) that enables systems to learn patterns from data and make predictions or decisions without being explicitly programmed.

1. What is Machine Learning?

Machine Learning is the study of algorithms that improve automatically through experience. Instead of relying on hard-coded instructions, ML systems identify patterns, adapt to new inputs, and improve their performance over time.

Example: A spam email filter learns to classify emails as “spam” or “not spam” by analyzing thousands of labeled emails.

2. Types of Machine Learning

  • Supervised Learning: Models learn from labeled data (input → output).
  • Unsupervised Learning: Models discover hidden patterns in unlabeled data.
  • Reinforcement Learning: An agent learns by interacting with an environment and receiving feedback (rewards or penalties).
Example:
  • Supervised: Predicting house prices based on size, location, etc.
  • Unsupervised: Grouping customers into clusters for marketing.
  • Reinforcement: A robot learning to walk by trial and error.

3. Hypothesis Space & Inductive Bias

The hypothesis space is the set of all possible models a learning algorithm can choose from. Inductive bias is the set of assumptions a model makes to predict unseen data.

Example: Linear regression assumes the relationship between variables is linear.

4. Model Evaluation

To assess how well a model performs, we use evaluation techniques such as:

  • Accuracy: Correct predictions ÷ Total predictions.
  • Cross-validation: Splitting data into multiple folds to ensure model generalization.
  • Precision, Recall, F1-score: Metrics for classification problems.
Example: In medical diagnosis, high recall ensures that fewer sick patients are missed.

5. Real-Life Applications of ML

Healthcare: Predicting disease risks from patient data, medical image analysis.
Finance: Fraud detection in credit card transactions, stock market prediction.
E-commerce: Personalized product recommendations (Amazon, Netflix, Spotify).
Transportation: Self-driving cars using reinforcement learning and computer vision.
Natural Language Processing (NLP): Voice assistants like Alexa, Google Translate.

6. Summary

Machine Learning is transforming industries by enabling data-driven decision-making. It covers different paradigms—supervised, unsupervised, and reinforcement learning—each applied to real-world problems. In the next lectures, we will dive into specific algorithms such as linear regression, decision trees, Bayesian learning, and support vector machines.

Lecture 17 Extension - Introduction to Machine Learning

Lecture 17 Extension: Introduction to Machine Learning (Part 2)

1. Why Machine Learning Works

Machine Learning (ML) is successful because it leverages data-driven patterns to make predictions or decisions without being explicitly programmed. Instead of hardcoding rules, models learn relationships between features (inputs) and labels (outputs).

Example: Spam detection does not rely on manually written rules like "if email contains 'lottery' mark as spam." Instead, an ML model learns from thousands of examples of spam and non-spam emails, automatically identifying patterns such as word frequency, sender, or links.

2. Mathematical Foundation

Key mathematical areas behind ML:

  • Linear Algebra: Used in vector/matrix operations for models like Linear Regression, PCA, Neural Networks.
  • Probability & Statistics: Basis of Bayesian Learning, Naïve Bayes, and probabilistic models.
  • Calculus: Optimization using derivatives and gradients (e.g., Gradient Descent).
  • Optimization: Finding parameters that minimize error functions, used in SVMs and Logistic Regression.
Example (Linear Regression): Suppose we want to predict house price based on size. Model: Price = β₀ + β₁ × Size Learning means estimating β₀ (intercept) and β₁ (slope) from data.

3. Types of Data in ML

  • Structured Data: Tables (e.g., sales records, medical data).
  • Unstructured Data: Images, text, audio, video.
  • Time-series Data: Stock prices, patient glucose levels, weather predictions.

4. Practical Workflow of ML Project

  1. Define the problem (classification, regression, clustering).
  2. Collect and clean data.
  3. Split into training, validation, and test sets.
  4. Choose an appropriate model (Linear, Tree-based, Neural Network, etc.).
  5. Train the model (optimize parameters).
  6. Evaluate using metrics (Accuracy, RMSE, F1-score, AUC).
  7. Deploy and monitor performance.

5. Applications of ML (Extended)

Healthcare: Predict diabetes using patient lifestyle and lab test results. Example: Logistic Regression can predict the probability of diabetes based on features like BMI, blood sugar, and family history.
Finance: Credit scoring and fraud detection. Example: SVMs can separate fraudulent vs. genuine transactions based on spending patterns.
Transportation: Self-driving cars use ML for object recognition, lane detection, and decision-making.
Recommender Systems: Collaborative filtering (e.g., Netflix, Amazon) suggests products or movies by learning from user-item interactions.

6. Hands-On Example (Python Pseudocode)


# Predicting house price with Linear Regression
from sklearn.linear_model import LinearRegression

X = [[650], [800], [1200], [1500]]  # house sizes
y = [70000, 85000, 125000, 150000]  # prices

model = LinearRegression()
model.fit(X, y)

print("Predicted price of 1000 sqft house:", model.predict([[1000]]))

7. Key Takeaways

  • ML is data-driven, adaptive, and improves with more examples.
  • Math (algebra, probability, calculus, optimization) forms the backbone.
  • Applications range from healthcare to finance, marketing, and AI systems.
  • Real-world ML involves careful preprocessing, evaluation, and deployment.
