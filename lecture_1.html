Lecture 1: Basic Matrix Theory

In this lecture, we introduce the fundamental concepts of matrices. A matrix is a rectangular array of numbers arranged in rows and columns. Matrices are widely used in mathematics, statistics, data science, and machine learning.

1. Definition of a Matrix

A matrix with m rows and n columns is called an m × n matrix:

A =
a11a12…a1n
a21a22…a2n
⋮⋮⋱⋮
am1am2…amn

2. Types of Matrices

  • Row Matrix: A matrix with only one row.
  • Column Matrix: A matrix with only one column.
  • Square Matrix: A matrix with equal number of rows and columns.
  • Diagonal Matrix: A square matrix where all off-diagonal elements are zero.
  • Identity Matrix: A diagonal matrix with all diagonal entries equal to 1.
  • Zero Matrix: A matrix where all elements are zero.

3. Matrix Operations

Some basic operations on matrices include:

  • Addition: Matrices of the same size can be added by adding corresponding elements.
  • Scalar Multiplication: Each element of a matrix is multiplied by a scalar.
  • Transpose: The rows become columns and columns become rows.
  • Matrix Multiplication: If A is (m × n) and B is (n × p), then AB is (m × p).
Example:
A =
12
34

B =
56
78

A + B =

68
1012

4. Applications of Matrices

  • Solving systems of linear equations
  • Transformations in computer graphics
  • Representation of data in statistics
  • Modeling in machine learning and AI
